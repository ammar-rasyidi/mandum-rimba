# Governance

Mandum Rimba is an open-source project with a single **steward** model: the code
is open to everyone, while direction and the brand stay with the maintainer. This
mirrors how single-vendor open-source projects (e.g. Grafana) are run, as opposed
to a neutral foundation (e.g. Kubernetes / CNCF).

## Roles

- **Maintainer / Steward — Ammar Rasyidi.** Owns the project direction, has final
  say on what ships, holds the **trademark and brand** (see [TRADEMARK.md](./TRADEMARK.md)),
  and is the release authority. Reachable at
  [@r.rasyidi](https://www.threads.com/@r.rasyidi).
- **Contributors.** Anyone who opens an issue or pull request. Trusted, regular
  contributors may be invited to a committer/reviewer role over time.

## How decisions are made

- Day-to-day: discussion happens in GitHub issues and pull requests. The
  maintainer decides, transparently, with community input.
- Editorial decisions follow the project's
  [principles](./README.md#editorial-principles-non-negotiable): evidence over
  accusation, every claim clickable, reproducible, never fabricate.
- Changes to data methodology are reflected in the in-app methodology changelog.

## Contributing

- Contributions are made by pull request and are licensed under
  **AGPL-3.0-or-later**, the project's code license.
- By submitting a contribution you agree to the
  [Contributor License terms](./CLA.md), so the project can keep its code under a
  single, consistent license and the steward can maintain the project (including
  future relicensing if ever needed).
- See [CONTRIBUTING.md](./CONTRIBUTING.md) for the practical workflow.

## The brand stays with the steward

The code is AGPL and belongs to the community to run, study, and improve. The
**name and logos** are not part of that grant; they remain the maintainer's, so
there is one clear, trustworthy "Mandum Rimba". Forks are welcome but must
rebrand (see [TRADEMARK.md](./TRADEMARK.md)).

## Future

If the project grows, the steward may move it under a non-profit foundation
(as Home Assistant did with the Open Home Foundation). Any such change, including
where the trademark lives, would be announced here. Until then, governance and
the brand rest with the maintainer.
